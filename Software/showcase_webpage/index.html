Project Description

The Smart Power Meter is a smart device that monitors power consumption without any external human intervention. The main goal of the project is to automate the entire process for electricity companies as well as enable customers of having possiblity to track their power consumption statistics and control the power mode status online. So, Smart Power Meter aims to automate the monthly billing process for companies and provide customers helpful visual data.

Components

ESP 32 Microcontroller

The ESP32 is a microcontroller development board that integrates Wi-Fi and Bluetooth connectivity along with a powerful processing unit. It is based on the ESP32 system-on-a-chip (SoC) designed by Espressif Systems.

PZEM-004T Sensor

The PZEM004T is an electrical power monitoring module that allows you to measure various parameters of an AC electrical system: Voltage, Current, Power, Energy, etc.

T92 Electromechanical Relay

An electromechanical relay is a switching device that uses an electromagnet to control electrical contacts, allowing or interrupting current flow. It has moving parts and is commonly used in control systems, power distribution, and telecommunications, but is slower and has a limited lifespan compared to solid-state relays.

Hi-Link 220V to 12V 220-12V Step Down Converter

A 220-12 step-down converter is an electrical device designed to convert higher voltage (220 volts) to a lower voltage (12 volts).

SIM800 GSM Module

SIM800L is a widely used module in the field of embedded systems and Internet of Things (IoT). It is a compact GSM/GPRS module that provides cellular connectivity for various applications.

SD Card SD Card Slot

An SD card module is an electronic module that allows the interfacing of SD (Secure Digital) memory cards with microcontrollers or other electronic systems. It provides a convenient way to read and write data to an SD card
